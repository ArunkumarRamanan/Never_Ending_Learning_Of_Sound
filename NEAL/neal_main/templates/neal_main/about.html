About About Humans identify sounds, able to classify them and draw relevancies. But how does a computer knows what a dog's bark sounds like? How does it knows that the incoming sound sounds like a bird? Can a computer learn all these just by browsing and classifying sounds on the Internet? NEAL believes it's possible! NELS (Never Ending Learning of Sound) is a computer program that runs 24x7 days to automatically extract aural knowledge from Internet data. It runs through the input audio files over web and tries to identify which sounds occur and when in the audio. It is an effort to build the world’s largest aural knowledge base with minimum human labeling effort – one that would be useful in several researches.
